Dad's Joke Generator <br>

Простое и весёлое приложение, которое загружает случайную шутку через API.  <br>
Каждое нажатие на кнопку — новая шутка. <br>

Функции <br>
- Получение рандомной шутки <br>
- Кнопка генерации <br>
- Анимация загрузки <br>
- Чистый и минималистичный интерфейс <br>

Используемые технологии <br>
- HTML  <br>
- CSS  <br>
- JavaScript  <br>
- Fetch API  <br>
- https://icanhazdadjoke.com API <br>
